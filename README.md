## Key features

- User registration & sign-in via Clerk (supports multiple accounts per user â€” see "two-account management")

- Create/read posts (text + images)

- Stories (ephemeral content)

- Follow / unfollow, connections, followers & following lists

- Notifications (in-app)

- Realtime 1:1 chat and typing indicators using Socket.IO

- Image uploads with ImageKit

- Simple admin/dev seeding using faker-js

- File uploads handled by multer

- Environment-driven config with dotenv

## Prerequisites

- Node.js >= 18 (recommended)

- npm

- MongoDB instance (Atlas)

- Clerk account (for auth)

- ImageKit account (for image CDN)

- (Optional) ngrok for local webhook testing
